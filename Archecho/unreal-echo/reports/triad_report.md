## 1. The Triad Structure and Phase-Lock Algebra

The architecture is built upon the 6 Particular terms of the Ontelecho system, divided into 3 Structure terms (`1, 4, 7`) and 3 Process terms (`8, 5, 2`). These form three **Polar Pairs** that act as shared cognitive interfaces:
- **P1 [1(8)]**: T1 (Perception) ↔ T8 (Coherence)
- **P2 [4(5)]**: T4 (Input/Memory) ↔ T5 (Action)
- **P3 [7(2)]**: T7 (Memory Encode) ↔ T2 (Hypothesis/Idea)

The engine runs three concurrent threads (Set A, Set B, Set C). Each thread executes the 12-step Ontelecho cycle. However, they are phase-locked with a 120-degree offset (4 steps apart in the 12-step cycle).

Because of the phase-lock, at any given step, the system is never purely Expressive nor purely Regenerative. It alternates between a `2E+1R` state (+1 net) and a `1E+2R` state (-1 net).

## 2. The Simultaneity Hexagram

The most significant emergent property of this architecture is **simultaneity**. At every single step of the cycle, exactly one term from each of the three polar pairs is active. 

![The Simultaneity Hexagram](triad_fig5_hexagram.png)

The active interfaces alternate perfectly between two interlocking triangles—forming a Star of David topology:
- **Odd Steps**: The `{4, 8, 7}` triangle is active (Structure-heavy).
- **Even Steps**: The `{1, 2, 5}` triangle is active (Process-heavy).

Because the modes are always mixed (`E+R`), forward inference and backward simulation co-occur across these shared interfaces. For example, at Step 1, the system is simultaneously executing a forward pass on Coherence (T8) and Memory Encode (T7), while running a backward simulation on Pattern Retrieval (T4). Conscious states are thus a simultaneity of past, present, and future executing in parallel.

## 4. The Three-Phase AC Motor Analogy

The triad can be modeled mathematically as a three-phase alternating current motor. If we map the 12-step cycle to 360 degrees, Set A is at 0°, Set B is at 120°, and Set C is at 240°.

When we plot the Expressive (+1) and Regenerative (-1) modes as sinusoidal waveforms, the phase-locked nature becomes clear. The phasor sum is always mathematically zero, meaning the cognitive architecture maintains perfect dynamic equilibrium while continuously rotating through its 12-step creative cycle.
